# Quiz Game 🎮

## 📌 Problem
Students need an engaging way to test their knowledge.

## 💡 Solution
This project is a Python-based quiz game that asks multiple-choice questions and calculates the user's score.

## ⚙️ Features
- Multiple-choice questions
- Score calculation
- Instant feedback
- Score saving

## 🛠️ Technologies Used
- Python
- File Handling

## ▶️ How to Run
1. Install Python
2. Run the following command:
   python main.py

## 📊 Example
User answers questions and gets a final score at the end.

- Learned Python basics
- Understood file handling
- Improved logical thinking
